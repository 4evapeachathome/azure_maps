Forget/Reset Password

Steps to Reset the Password

Enter the valid Emailid/Username and Click on Forget Password, the confirmation Pop up will appear.

The user needs to click on Forget Password link by providing the valid email id, otherwise the vallidation will appear.

CMS Admin Login

1.Open the CMS site. User login
2.Open the Content Manager from the left menu.
3.Navigate to User login menu.
4.Select the specific user who gave the request for Password Reset.
5.Temp.Password should generate automatically.
6.Click on Publish .
7.Now the user will receive an email with: Username, New temp_password and Forgot password page link (same as the sign-in page).

Reset Password

1. The User should open the received email.
2. Click on the link provided in the email.
3. The Login page for reset the password will be appeared
Login login
4. The User needs to enter the new temp_password and set the new password.

5.The Confirmation alert should appear.
6. A password updated email will send to the user.
